# 固体物理基础 (Solid State Physics)

## 简介
研究晶体中原子排列和电子行为。核心是能带理论、晶格振动和电子输运。

## 关键公式
- Bloch 定理: $\psi_k(\mathbf{r}) = u_k(\mathbf{r}) e^{i\mathbf{k} \cdot \mathbf{r}}$
- 近自由电子近似能带: $E(\mathbf{k}) = \frac{\hbar^2 k^2}{2m} + \sum_{\mathbf{G}} |V_\mathbf{G}|^2 \frac{2m}{\hbar^2} \frac{1}{k^2 - |\mathbf{k} - \mathbf{G}|^2}$
- 声子色散 (单原子链): $\omega = 2\sqrt{\frac{K}{M}} |\sin(ka/2)|$
- Drude 电导率: $\sigma = \frac{ne^2 \tau}{m}$
