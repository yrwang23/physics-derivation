# User-Added Knowledge

This directory stores knowledge entries you add through interactions with the physics derivation skill.

## Adding a New Entry

### Option 1: During Derivation Q&A
When the skill explains a new concept you want to save, it will automatically offer to add it to your knowledge base.

### Option 2: Using the Script

Then paste the content in markdown format.

### Option 3: Manual
Create a new .md file and add a row to INDEX.md in the User-Added Knowledge section.
