# 晶体X射线衍射原理 (X-ray Diffraction from Crystals)

## 简介
X 射线波长与晶面间距量级相同，被周期性原子阵列散射。布拉格模型从实空间指出光程差满足波长整数倍时产生相干极大。劳厄模型从倒空间指出入射与散射波矢差等于倒格矢时发生相长干涉。两套理论等价。

## 关键公式
- 布拉格条件: $2d\sin\theta = n\lambda$
- 劳厄条件: $\Delta\vec{k} = \vec{k}' - \vec{k} = \vec{K}_n$
- 等价推导:
  $2\frac{2\pi}{\lambda}\sin\theta = \frac{2\pi}{d}n \implies 2d\sin\theta = n\lambda$
