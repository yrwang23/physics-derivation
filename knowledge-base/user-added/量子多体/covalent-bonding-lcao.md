# 共价结合与LCAO模型 (Covalent Bonding & LCAO Model)

## 简介
共价晶体的核心在于电子云重叠和自旋反平行配对。LCAO 方法假设分子轨道是孤立原子波函数的线性叠加，通过求解久期方程得到成键轨道（能量低）和反键轨道（能量高）。成键轨道增加核间电子云密度，降低系统能量。

## 关键公式
- 分子轨道: $\psi = C_A \varphi_A + C_B \varphi_B$
- 成键/反键能量:
  $\varepsilon^{\pm} \approx \varepsilon_0 + \frac{t \pm \beta}{1 \pm S}$
- 有效电荷:
  $q_B^* = 3 - 8\frac{\lambda^2}{1+\lambda^2}$
