# 离子晶体结合与体变模量 (Ionic Bonding & Bulk Modulus)

## 简介
离子晶体由正负离子静电吸引结合。总结合能由长程引力（马德隆能）和短程排斥力（泡利原理）构成。长程作用通过马德隆常数求和，短程作用用幂函数近似。系统在受力平衡时总能量达极小值，对能量关于体积的二阶导数可求出体变模量。

## 关键公式
- 马德隆电势能:
  $E = \frac{1}{2} \frac{\alpha q^2}{4\pi\varepsilon_0 r}$
- 总结合能:
  $U = N\left[ -\frac{\alpha q^2}{4\pi\varepsilon_0 r} + \frac{b}{r^n} \right]$
- 平衡结合能:
  $W = \frac{N\alpha q^2}{4\pi\varepsilon_0 r_0}\left(1 - \frac{1}{n}\right)$
- 体变模量:
  $k = \frac{(n-1)}{18 r_0^4} \frac{\alpha q^2}{4\pi\varepsilon_0}$
