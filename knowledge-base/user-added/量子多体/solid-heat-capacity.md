# 固体热容模型 (Heat Capacity of Solids)

## 简介
经典能均分定理预言热容为常数 $3NK$，与低温实验相悖。爱因斯坦模型假设独立简谐振子，解释热容冻结但仍低温衰减过快。德拜模型将固体视为连续弹性介质，线性色散加截止频率，成功解释低温 $T^3$ 规律。

## 关键公式
- 经典热容（高温极限）: $C_v = 3NK$
- 爱因斯坦公式:
  $C_V = 3NK(\Theta_E/T)^2 \frac{e^{\Theta_E/T}}{(e^{\Theta_E/T}-1)^2}$
- 德拜低温: $C_V \propto T^3$
