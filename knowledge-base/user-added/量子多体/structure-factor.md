# 结构因子与系统消光 (Structure Factor & Systematic Absences)

## 简介
即使满足劳厄条件，某些衍射斑点也可能因晶胞内不同位置原子的散射波反相干涉而消失（系统消光）。结构因子定量描述晶胞内原子分布对衍射强度的影响。BCC 和 FCC 的消光规律是推断未知晶体结构的实验依据。

## 关键公式
- 结构因子:
  $F_{hkl} = \sum_j f_j e^{-i2\pi(hx_j + ky_j + lz_j)}$
- BCC 消光: $h+k+l$ 为奇数时 $F=0$
- FCC 消光: $h,k,l$ 奇偶混杂时 $F=0$，全奇或全偶时存在
