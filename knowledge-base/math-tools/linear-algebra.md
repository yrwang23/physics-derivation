# 线性代数 (Linear Algebra)

## 简介
向量空间和线性映射的数学理论。量子力学的数学基础，也是许多物理计算的核心工具。

## 关键公式
- 本征值方程: $\mathbf{A} \mathbf{v} = \lambda \mathbf{v}$
- 内积: $\langle u | v \rangle = \sum_i u_i^* v_i$
- 张量积: $|\psi\rangle \otimes |\phi\rangle = |\psi\phi\rangle$
- 谱分解: $\hat{A} = \sum_i a_i |a_i\rangle\langle a_i|$
