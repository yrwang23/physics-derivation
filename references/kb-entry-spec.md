# 知识条目输入规范

将此文档提供给帮你整理笔记的 AI，让它按以下格式从 PDF 笔记中提取知识点。

---

## 整体要求

1. **每个知识点一个文件** — 提取一个完整的知识点
2. **使用以下模板** — 严格按模板格式输出
3. **LaTeX 公式** — 用 $...$（行内）和 $$...$$（独立行）包裹
4. **文件名** — 英文小写加连字符，如 `bcs-theory.txt`
5. **编码** — UTF-8

## 模板

```
---
name: "Entry Name"
type: theory                 # theory 或 math-tool
category: "量子多体"          # 所属分类：量子多体 / 量子信息与神经网络 / 凝聚态计算 / 其他
topics: ["topic1", "topic2"]
prerequisites:
  - "Prerequisite 1"
---

# Entry Name

## 简介
2-5 句中文描述。

## 关键公式
- [含义]: $formula$
- [含义]: $$formula$$

## 补充说明（可选）
```

## 字段说明

### category
当前支持四个分类：
- `量子多体` — 多体理论工具
- `量子信息与神经网络` — 量子信息、量子计算、神经网络量子态等
- `凝聚态计算` — DMFT、QMC、DMRG、DFT 等数值方法
- `其他` — 不属于以上分类的知识

## 使用流程

1. 你有 PDF 笔记
2. 把本文档给其他 AI
3. 那个 AI 按模板提取知识点，每个一个 .txt 文件
4. 发回给我
5. 我导入知识库对应分类目录，更新 INDEX.md
